# Claude Code レクチャー 事前準備チェックリスト

レクチャー当日までに以下を完了してください。所要時間は約15〜30分です。

## 必須

- [ ] **Node.js のインストール**
  - https://nodejs.org/ からLTS版をダウンロード・インストール
  - 確認: ターミナルで `node -v` → v18以上が表示されればOK

- [ ] **Claude Code のインストール**
  - ターミナルで以下を実行:
    ```
    npm install -g @anthropic-ai/claude-code
    ```
  - 確認: `claude --version` → バージョン番号が表示されればOK

- [ ] **Anthropic Team アカウントにログイン**
  - ターミナルで `claude` を実行
  - ブラウザが開くので、招待されたTeamアカウントでログイン
  - ターミナルに戻ってClaude Codeが起動していればOK

- [ ] **VS Code のインストール**
  - https://code.visualstudio.com/ からダウンロード・インストール
  - ターミナルの開き方: 上部メニュー「ターミナル」→「新しいターミナル」

- [ ] **Git のインストール確認**
  - 確認: ターミナルで `git --version` → バージョン番号が表示されればOK
  - 入っていない場合: https://git-scm.com/ からインストール

## 困ったら

- エラーが出た場合はスクリーンショットを撮って、Slackで角田さんに送ってください
- 当日もサポートしますので、できなくても大丈夫です
