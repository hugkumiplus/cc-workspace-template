# Claude Code コマンド＆ショートカット集

## 基本コマンド

| コマンド | 説明 |
|---------|------|
| `claude` | Claude Codeを起動 |
| `/help` | 使えるコマンド一覧 |
| `/clear` | 会話履歴をリセット |
| `/compact` | 会話を要約して圧縮 |
| `/doctor` | 環境診断 |
| `/status` | 現在の設定確認 |
| `/model` | AIモデルの確認・変更 |
| `/config` | 設定変更（言語の日本語化等） |
| `/mcp` | MCP接続の管理 |
| `/plan` | 計画だけ先に見せてもらうモード |
| `/cost` | セッションのコスト確認 |
| `/permissions` | 権限設定の確認・変更 |

## 便利コマンド

| コマンド | 説明 |
|---------|------|
| `/btw` | タスク中に会話を汚さず質問を差し込める |
| `/rewind`（Esc×2） | コードと会話を巻き戻し |
| `/branch` | 会話を分岐して並列試行 |
| `/remote-control`（/rc） | スマホでセッションをリアルタイム同期 |
| `/insights` | 過去1ヶ月の使用パターンをHTMLレポート化 |
| `/export` | 会話全体をMarkdownでエクスポート |
| `/loop` | 定期タスク実行（デフォルト10分間隔） |
| `/schedule` | リモートエージェントの定期実行 |
| `/fast` | 高速出力モード切替 |
| `/model opusplan` | 設計はOpus、実装はSonnetに自動切替 |

## ファイル参照

| 操作 | 方法 |
|------|------|
| ファイルを参照して質問 | `@path/to/file` と入力 |
| 複数ファイルを同時参照 | `@file1 @file2` |

## ショートカット

| キー | 説明 |
|------|------|
| `Ctrl+C` | 実行中の処理を止める（最重要！） |
| `Ctrl+V` | スクショ直接ペースト |
| `Ctrl+J` / `Option+Enter` | ターミナル内改行 |
| `Shift+Enter` | 複数行入力（送信しない） |
| `Ctrl+R` | プロンプト履歴検索 |
| `Ctrl+U` | 入力行一括削除 |
| `Shift+Tab` | モデル切替（Opus↔Sonnet） |
| `↑` キー | 前の入力を呼び出す |
| `Tab` | コマンド・ファイル名の補完 |
| `Esc` | 生成を中断 |

## ターミナル基本コマンド（必須5つ）

| コマンド | 読み方 | 何をするか |
|---|---|---|
| `pwd` | ピーダブリューディー | 今いる場所を表示 |
| `ls` | エルエス | フォルダの中身を見る |
| `cd フォルダ名` | シーディー | フォルダを移動 |
| `cd ..` | シーディー ドットドット | 一つ上に戻る |
| `mkdir フォルダ名` | エムケーディーアール | 新しいフォルダを作る |

## 緊急脱出

| 状況 | 操作 |
|------|------|
| 止まらない | `Ctrl + C`（1〜3回押す） |
| 全部消したい | `Ctrl + C` → `exit` |
| 文字が変になった | `reset` と入力 |

> ターミナルで「壊れる」ことはほぼない。安心してください。
