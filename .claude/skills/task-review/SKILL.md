# タスク振り返りワークフロー

## 概要

「振り返りなき完了なし」の原則に従い、タスクの成果を評価し、**成功パターンも失敗パターンも**学びとして蓄積し、システム全体を改善する。PDCAの Check + Act。

## ワークフロー

### Step 1: タスク状況の確認（Automatic）

対象タスクフォルダの以下を読み込む:

1. **goal.md** — 目的・ゴール・理想・要件
2. **tasks.md** — TODO の完了状況
3. 成果物（output/ や作業ファイル）

確認結果を表示:

> **タスク振り返り: {タスク名}**
>
> ゴール達成度:
> - {ゴール1}: ✅ 達成 / ⚠️ 部分的 / ❌ 未達成
>
> TODO進捗: {完了数}/{総数}
> - 未完了: {残りのTODOリスト}

### Step 2: ユーザーとの振り返り対話（Interactive）

`AskUserQuestion` で以下を聞く:

> 振り返りをしましょう。
>
> 1. **良かった点** — うまくいったこと、今後も続けたいこと
> 2. **うまくいったアプローチ** — 次回も再利用できるやり方・コツ
> 3. **改善点** — こうすればもっと良かったこと
> 4. **気づき** — 今回学んだこと、次に活かしたいこと
>
> 自由に話してください。一つずつでも大丈夫です。

深掘りが必要なら追加で質問する。特に以下を引き出す:
- Claudeの振る舞いで良かった/悪かった点
- ワークフローで改善したい点
- **うまくいったアプローチ（成功パターン）** — これを明示的に聞く

### Step 3: review.md の記録（Automatic）

振り返り内容を review.md に記録:

```markdown
## 振り返り
実施日: {YYYY-MM-DD}

### ゴール達成度
{Step 1の評価結果}

## 良かった点
{ユーザーの回答}

## うまくいったアプローチ（再利用可能）
<!-- 次回以降も使えるパターンを明示的に記録 -->
{成功パターンを具体的に記述}

## 改善点
{ユーザーの回答}

## 学び
{ユーザーの回答 + Claudeの分析}
```

### Step 4: Memory への学習保存（Automatic）

振り返りから**横断的に使える学び**を抽出し、Memory に保存する。

**成功パターンも失敗パターンも両方保存する:**
- **成功パターン**: 「このアプローチがうまくいった」→ 次回同様の状況で再利用
- **失敗パターン**: 「これは避けるべき」→ 同じミスの防止

保存判断基準:
- **保存する**: 今後の別タスクにも適用できるパターン・教訓
- **保存しない**: このタスク固有の事実、コードから読み取れること

保存前にユーザーに確認:

> 以下の学びを Memory に保存してよいですか？
> - {成功パターン} → feedback型: {ファイル名}
> - {改善点} → feedback型: {ファイル名}
>
> これにより、次回以降のタスクで自動的に反映されます。

### Step 5: 改善提案（Interactive）

振り返りから以下を検討・提案する:

1. **Skill化の提案**: 繰り返し使われるワークフローがあれば新Skill作成を提案
2. **CLAUDE.md の更新提案**: 原則や構成ルールの追加・修正が必要なら提案
3. **既存Skillの改善提案**: /task-start, /task-review 自体の改善

> **改善提案:**
> {提案があれば具体的に記載。なければ「現時点で改善提案はありません」}
>
> 実施しますか？

ユーザーが承認したら即座に実行する。

### Step 6: タスク完了処理（Automatic）

1. tasks.md の全TODOが完了していれば、ステータスを完了に更新
2. 未完了TODOがあれば、対応を確認（持ち越し/取り下げ）

> タスク「{タスク名}」の振り返りが完了しました。
> お疲れさまでした！

## 重要な注意事項

- review.md は事実ベースで書く。感情的・主観的な評価は避ける
- **成功パターンを明示的に聞き出す** — 失敗だけでなく「うまくいったアプローチ」も必ず記録する
- Memory 保存は必ずユーザー確認を取る
- 改善提案は具体的かつ実行可能なものに限る
- 改善提案が空振りでも構わない。提案しないよりは提案する方がよい
- /task-review 自体の改善フィードバックも歓迎する（メタ改善）
